# ТЗ для Figma Make: ПКО-300

## Продукт

**ПКО-300** — рейтинг коллекторских агентств России. Одностраничный веб-дашборд (десктоп 1440px) с интерактивной таблицей и боковой панелью фильтров. 545 компаний.

## Главная таблица — колонки

| # | Колонка | Формат | Пояснение |

| 1 | **↑↓ %** | Стрелка + процент, зелёный/красный | Изменение позиции в рейтинге относительно прошлого года (YoY по сумме Выручка + Прочие доходы) |

| 8 | **Стаж** | лет | Сколько лет на рынке |

## Панель фильтров (левый сайдбар)

Фильтры сгруппированы по разделам. Каждая группа — сворачиваемая секция. При выборе фильтра таблица мгновенно обновляется + пересчитываются карточки-сводки наверху.

---

### ГРУППА 1: «Финансовая отчётность»

**Переключатель года:** `2020` | `2021` | `2022` | `2023` | `2024` (по умолчанию)

При выборе года в таблицу добавляются дополнительные колонки:

| Колонка | Данные | Источник |
|---------|--------|----------|
| Выручка | тыс ₽ | finances.json → Выручка (тыс руб) |
| Расходы | тыс ₽ | finances.json → Расходы (тыс руб) |
| Прибыль | тыс ₽ | finances.json → Прибыль (тыс руб) |
| Активы | тыс ₽ | finances.json → Активы (тыс руб) |
| Дебиторка | тыс ₽ | finances.json → Дебиторка (тыс руб) |

**Фильтры-ползунки:**
- Выручка: min–max
- Прибыль: min–max
- Активы: min–max

**Переключатели:**
- Прибыльность: `Все` | `Прибыльные (394)` | `Убыточные (116)`
- Динамика YoY: `Все` | `Растут ↑` | `Падают ↓`

**Кнопка «Показать динамику»** — при нажатии рядом с каждой цифрой появляется Δ% к предыдущему году

## Визуальный стиль

- Финтех-дашборд, чистый, минималистичный
- Светлая тема
- Шрифт: моноширинный для цифр (tabular nums), без засечек для текста
- Числа ВСЕГДА с разделителями тысяч (пробел): `15 819 586`
- Валюта: тыс ₽
- Логотипы: круглая маска 32×32px, если нет лого — первая буква названия на цветном фоне
- Стрелки рейтинга: ↑ зелёная, ↓ красная, — серая
- Строки таблицы: зебра (чередование белый/серый), hover-подсветка

## Интерактивность

- **Сортировка**: клик по заголовку любой колонки → сортировка ↑↓
- **Фильтры**: мгновенное обновление таблицы + карточек-сводок
- **Поиск**: по названию или ИНН, live-search
- **Экспорт**: кнопка «Скачать CSV» — выгружает отфильтрованные данные
- **Счётчик**: «Найдено: 127 из 545 компаний» — обновляется при фильтрации

## Ключевые правила для Figma Make

1. **Данные РЕАЛЬНЫЕ** — использовать строки из раздела «ТОЧНЫЕ ДАННЫЕ», не placeholder
2. **Рейтинг по сумме** Выручка + Прочие доходы, показывать одним числом в колонке
3. **Фильтры динамически меняют колонки** — при выборе «Финансовая отчётность» добавляются расходы/активы/дебиторка; при «Облигации» — ISIN/купон/объём
4. **Логотипы** — круглые, 32×32, берутся из `data/logos/` по маппингу `logo_mapping.json`
5. Десктоп 1440px, одна страница
6. Фильтры — левый сайдбар ~300px, таблица справа
7. 4 информационные карточки — над таблицей
