# ТЗ для Figma Make: ПКО-300

## Продукт

**ПКО-300** — рейтинг коллекторских агентств России для частных инвесторов. Одностраничный веб-дашборд (десктоп 1440px) с интерактивной таблицей. 545 компаний. Все фильтры — строго горизонтально сверху, никакого левого сайдбара.

---

## Структура страницы (сверху вниз)

```
┌──────────────────────────────────────────────────────────────────┐
│  ПКО-300                              [🔍 Поиск...]  [CSV ↓]   │  ← Шапка
├──────────────────────────────────────────────────────────────────┤
│  [Рейтинг]    [Финансы]    [Инвестиции]                         │  ← 3 таба
├──────────────────────────────────────────────────────────────────┤
│  Фильтр 1 | Фильтр 2 | Фильтр 3 | ...                         │  ← Фильтры (зависят от таба)
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Таблица на всю ширину (колонки зависят от таба)                │
│  15-20 строк на экране, компактные строки                       │

└──────────────────────────────────────────────────────────────────┘
```

**Нет:** левого сайдбара, карточек-сводок, счётчика «Найдено», пресетов-чипов.

## Визуальный стиль

- Финтех-дашборд, светлый, минималистичный
- Шрифт: моноширинный для цифр (tabular nums), без засечек для текста
- Числа ВСЕГДА с разделителями тысяч пробелом: `15 819 586`
- Валюта: тыс ₽
- Логотипы: круглая маска 32×32, если нет — первая буква на цветном фоне
- Строки: компактные (15-20 на экране), зебра, hover-подсветка
- Фильтры — строго горизонтально, никакого левого сайдбара

## Интерактивность

- **Сортировка**: клик по заголовку колонки → ↑↓
- **Фильтры**: мгновенное обновление таблицы
- **Поиск**: по названию или ИНН, live-search

- **Количество фреймов**: минимум 3 (по одному на таб: Рейтинг, Финансы — Доходы и расходы, Инвестиции — Облигации)

- **Правило приоритета кружков ●**: если компания одновременно 🟢 и 🔵, показывать 🟢 (публичное важнее)
- **Числовой формат**: все суммы с пробелом-разделителем тысяч (15 819 586), без десятичных для целых чисел
