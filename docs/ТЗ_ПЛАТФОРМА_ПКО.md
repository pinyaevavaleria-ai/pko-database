# ТЗ: Аналитическая платформа «Рейтинг ПКО-300»

## 1. Цель продукта

Платформа для инвесторов, позволяющая анализировать 545 профессиональных коллекторских организаций (ПКО) России через интерактивные фильтры и финансовую отчётность. Помогает принять решение: в какую компанию инвестировать и каким способом.

---

## 2. Источник данных

JSON-файлы из GitHub-репозитория (`data/export/`):

| Файл | Записей | Содержание |
|------|---------|-----------|
| `companies.json` | 545 | Карточки компаний (ИНН, ОГРН, стаж, сайт, рейтинг) |
| `rating.json` | 545 | Рейтинг по доходу (выручка + прочие доходы, YoY%) |
| `finances.json` | 2456 | Финансы по годам 2020-2024 (выручка, расходы, прибыль, активы) |
| `capital_structure.json` | 545 | Структура капитала (D/E, заёмный/собственный) |
| `management.json` | 545 | Руководство и учредители |
| `bonds.json` | 32 | Облигации (купон, ISIN, рейтинг, объём) |
| `fundraising.json` | 23 | Способы привлечения капитала |

## 3. Структура платформы

### 3.1. Главная страница — Рейтинг ПКО-300

**Основной элемент**: таблица-рейтинг всех 545 компаний, отсортированных по совокупному доходу.

#### Колонки таблицы:

| # | Колонка | Данные | Визуал |
|---|---------|--------|--------|
| 1 | **№ рейтинга** | 1-545 | Число в кружке |
| 2 | **Название** | Краткое название ПКО | Текст, клик → карточка компании |
| 3 | **Совокупный доход 2024** | Выручка + Прочие доходы (тыс руб) | Число с разделителями |
| 4 | **Динамика YoY** | % изменения к 2023 | ↑ зелёная стрелка / ↓ красная стрелка + % |
| 5 | **Прибыль 2024** | Чистая прибыль (тыс руб) | Зелёный если >0, красный если <0 |
| 6 | **Маржа прибыли** | Прибыль/Выручка × 100% | % с цветом |
| 7 | **D/E ratio** | Заёмный/Собственный капитал | Цвет: ≤1 зелёный, 1-3 жёлтый, >3 красный |
| 8 | **Стаж** | Лет на рынке | Число |
| 9 | **Как инвестировать** | Бейдж типа | 🟢 Облигации / 🟢 Займы / 🟡 Корпоративный / ⚪ — |

#### Панель фильтров (слева или сверху):

**Группа «Финансы»:**
- Выручка 2024: ползунок min-max (от 0 до 17.5 млрд руб)
- Прибыльность: переключатель [Все | Прибыльные (394) | Убыточные (116)]
- Динамика YoY: переключатель [Все | Растут ↑ | Падают ↓]
- Маржа прибыли: ползунок min-max (%)

**Группа «Надёжность»:**
- D/E ratio: сегменты [≤1 здоровый (93) | 1-3 умеренный (27) | >3 высокий (8) | Все]
- Стаж: сегменты [<3 года (63) | 3-5 лет (119) | 5-10 лет (210) | >10 лет (152)]

**Группа «Размер»:**
- Сегмент: чекбоксы [Микро <10 млн (146) | Малые 10-100 млн (200) | Средние 100 млн-1 млрд (120) | Крупные >1 млрд (28)]

**Группа «Инвестору»:**
- Способ инвестирования: чекбоксы
  - 🟢 Облигации на бирже (10 компаний)
  - 🟢 Займы через сайт (3 компании)
  - 🟡 Корпоративная структура (9 компаний)
  - ⚪ Не привлекают / нет данных (523 компании)
- Кредитный рейтинг: [Все | A и выше | BBB | BB | B | Без рейтинга]
- Есть облигации: [Да | Нет | Все]

**Группа «Финансовая отчётность» (расширенные фильтры):**
- Активы: ползунок min-max
- Дебиторская задолженность: ползунок min-max
- Собственный капитал: ползунок [>0 | <0 (отрицательный) | Все]
- Расходы: ползунок min-max

**Быстрые пресеты фильтров (кнопки сверху):**
- «Топ-50 по доходу» — первые 50 из рейтинга
- «Надёжные» — D/E ≤ 1, стаж > 5 лет, прибыльные
- «Растущие» — YoY > 50%, прибыльные
- «Можно инвестировать» — есть облигации или займы через сайт
- «Крупнейшие» — выручка > 1 млрд

**Счётчик результатов:**
- «Найдено: 127 из 545 компаний» — обновляется при каждом фильтре

## 4. Рассчитываемые показатели (формулы)

Эти метрики рассчитываются на лету из имеющихся данных:

```
Маржа прибыли = (Прибыль / Выручка) × 100%
ROE = (Прибыль / Собственный капитал) × 100%  [если собств. капитал > 0]
ROA = (Прибыль / Активы) × 100%
Доля рынка = (Выручка компании / 157,300,561) × 100%
CAGR = (Выручка_2024 / Выручка_2020)^(1/4) - 1  [если есть данные за оба года]
Совокупный доход = Выручка + Прочие доходы
YoY = (Доход_2024 - Доход_2023) / |Доход_2023| × 100%
```

## 5. Цветовая кодировка

| Элемент | Зелёный | Жёлтый | Красный | Серый |
|---------|---------|--------|---------|-------|
| YoY динамика | > 0% (рост) | — | < 0% (падение) | нет данных |
| Прибыль | > 0 | — | < 0 (убыток) | нет данных |
| D/E ratio | ≤ 1 | 1 - 3 | > 3 | нет данных |
| Маржа | > 20% | 5-20% | < 5% | нет данных |
| Стаж | > 10 лет | 5-10 лет | < 3 года | — |

## 6. Интерактивность

- **Сортировка**: клик по заголовку колонки → сортировка по убыванию/возрастанию
- **Фильтры**: мгновенное обновление таблицы при изменении фильтра
- **Поиск**: строка поиска по названию компании или ИНН
- **Экспорт**: кнопка «Скачать отфильтрованные данные» (CSV)
- **Сравнение**: выбрать 2-3 компании → сравнить показатели бок о бок

---

## 7. Адаптивность

- **Десктоп**: полная таблица + фильтры слева
- **Планшет**: таблица с горизонтальным скроллом, фильтры сверху (сворачиваемые)
- **Мобильный**: карточки вместо таблицы, фильтры в модальном окне
