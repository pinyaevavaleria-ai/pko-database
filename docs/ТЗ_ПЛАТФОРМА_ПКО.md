# ТЗ: Аналитическая платформа «ПКО-300»

## 1. Цель продукта

Платформа для **частных инвесторов**, позволяющая анализировать 545 профессиональных коллекторских организаций (ПКО) России. Помогает принять решение: в какую компанию инвестировать и каким способом.

---

## 2. Источник данных

JSON-файлы из GitHub-репозитория (`data/export/`):

| Файл | Записей | Содержание |
|------|---------|-----------|
| `companies.json` | 545 | Карточки компаний (ИНН, ОГРН, стаж, сайт, регион, НАПКА, риск ФССП) |
| `rating.json` | 545 | Рейтинг по доходу (выручка + прочие доходы, YoY%) |
| `finances.json` | 2456+ | Финансы по годам 2020-2024 (16 метрик, см. раздел 4) |
| `capital_structure.json` | 545 | Структура капитала (D/E, заёмный/собственный) |
| `management.json` | 545 | Руководство и учредители |
| `bonds.json` | 32 | Облигации (купон, ISIN, рейтинг, объём) |
| `fundraising.json` | 23 | Способы привлечения капитала |
| `logo_mapping.json` | 446 | Маппинг ИНН → логотип |

## 3. Структура платформы — ОДНА СТРАНИЦА, 3 ТАБА

Одна страница с интерактивной таблицей. **Все фильтры — строго горизонтально сверху.** Никакого левого сайдбара, никаких карточек-сводок, никаких пресетов-чипов.

### Лейаут

```
┌──────────────────────────────────────────────────────────┐
│  ПКО-300                        [🔍 Поиск]  [CSV ↓]     │  Шапка
├──────────────────────────────────────────────────────────┤
│  [Рейтинг]   [Финансы]   [Инвестиции]                   │  3 таба
├──────────────────────────────────────────────────────────┤
│  Фильтр 1 | Фильтр 2 | Фильтр 3 | ...                  │  Фильтры (меняются по табу)
├──────────────────────────────────────────────────────────┤
│  Таблица на всю ширину, 15-20 строк на экране           │
│  Колонки зависят от активного таба                      │
└──────────────────────────────────────────────────────────┘
```

### Таб «Рейтинг» (по умолчанию)

545 компаний, отсортированных по сумме (Выручка + Прочие доходы).

| # | Колонка | Данные | Визуал |
|---|---------|--------|--------|
| 1 | **↑↓ %** | Δ к 2023 | Зелёная ↑ / красная ↓ стрелка + % |
| 2 | **№** | 1-545 | Число в кружке |
| 3 | **Лого** | Логотип | Круг 32×32, если нет — первая буква |
| 4 | **Компания** | Краткое название | Текст |
| 5 | **Выручка + пр. доходы** | тыс руб | Число с разделителями |
| 6 | **Чистая прибыль** | тыс руб | Зелёный >0, красный <0 |
| 7 | **CAGR** | % за 5 лет | Цвет по шкале |
| 8 | **Стаж** | Лет | Число |
| 9 | **●** | Тип привлечения | Цветной кружок с тултипом |

**Колонка ●** — цветные кружки:
- 🟢 Публичное привлечение (облигации / займы через сайт) — тултип: «Публичное привлечение: облигации / займы»
- 🔵 Корпоративное — тултип: «Инвестируется материнской структурой»
- ⚪ Нет данных — тултип: «Нет данных о привлечении капитала»

**Фильтры таба «Рейтинг»:**

## 5. Рассчитываемые показатели

```
Совокупный доход = Выручка + Прочие доходы
YoY = (Доход_2024 - Доход_2023) / |Доход_2023| × 100%
CAGR = (Выручка_2024 / Выручка_2020)^(1/4) - 1
Маржа прибыли = Чистая прибыль / Выручка × 100%
Валовая маржа = Валовая прибыль / Выручка × 100%
D/E ratio = Заёмный капитал / Собственный капитал
```

## 6. Цветовая кодировка

| Элемент | Зелёный | Жёлтый | Красный | Серый |
|---------|---------|--------|---------|-------|
| YoY динамика | > 0% | — | < 0% | нет данных |
| Прибыль | > 0 | — | < 0 | нет данных |
| D/E ratio | ≤ 1 | 1-3 | > 3 | нет данных |
| CAGR | > 20% | 0-20% | < 0% | нет данных |
| Привлечение | 🟢 публичное | — | 🔵 корпоративное | ⚪ нет данных |
