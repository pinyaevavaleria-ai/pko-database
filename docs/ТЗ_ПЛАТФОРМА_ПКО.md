# ТЗ: Аналитическая платформа «Рейтинг ПКО-300»

## 1. Цель продукта

Платформа для инвесторов, позволяющая анализировать 545 профессиональных коллекторских организаций (ПКО) России через интерактивные фильтры и финансовую отчётность. Помогает принять решение: в какую компанию инвестировать и каким способом.

---

JSON-файлы из GitHub-репозитория (`data/export/`):

| `companies.json` | 545 | Карточки компаний (ИНН, ОГРН, стаж, сайт, рейтинг) |
| `rating.json` | 545 | Рейтинг по доходу (выручка + прочие доходы, YoY%) |

## 3. Структура платформы — ОДНА СТРАНИЦА

Вся платформа — это **одна страница** с интерактивной таблицей и фильтрами. Нет отдельных страниц или карточек компаний. Пользователь управляет фильтрами → таблица мгновенно обновляется, показывая нужные данные.

## 4. Рассчитываемые показатели (формулы)

Эти метрики рассчитываются на лету из имеющихся данных:

```
Маржа прибыли = (Прибыль / Выручка) × 100%
ROE = (Прибыль / Собственный капитал) × 100%  [если собств. капитал > 0]
ROA = (Прибыль / Активы) × 100%
Доля рынка = (Выручка компании / 157,300,561) × 100%
CAGR = (Выручка_2024 / Выручка_2020)^(1/4) - 1  [если есть данные за оба года]
Совокупный доход = Выручка + Прочие доходы
YoY = (Доход_2024 - Доход_2023) / |Доход_2023| × 100%
```

## 5. Цветовая кодировка

| Элемент | Зелёный | Жёлтый | Красный | Серый |
|---------|---------|--------|---------|-------|
| YoY динамика | > 0% (рост) | — | < 0% (падение) | нет данных |
| Прибыль | > 0 | — | < 0 (убыток) | нет данных |
| D/E ratio | ≤ 1 | 1 - 3 | > 3 | нет данных |
| Маржа | > 20% | 5-20% | < 5% | нет данных |
| Стаж | > 10 лет | 5-10 лет | < 3 года | — |

## 6. Интерактивность

- **Сортировка**: клик по заголовку колонки → сортировка по убыванию/возрастанию
- **Фильтры**: мгновенное обновление таблицы при изменении фильтра
- **Поиск**: строка поиска по названию компании или ИНН
- **Экспорт**: кнопка «Скачать отфильтрованные данные» (CSV)
- **Сравнение**: выбрать 2-3 компании → сравнить показатели бок о бок

---

## 7. Адаптивность

- **Десктоп**: полная таблица + фильтры слева
- **Планшет**: таблица с горизонтальным скроллом, фильтры сверху (сворачиваемые)
- **Мобильный**: карточки вместо таблицы, фильтры в модальном окне
